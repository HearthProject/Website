Our first project is the Hearth Launcher, a launcher and pack installer for modded Minecraft that has been designed from the ground up to provide the most commonly required features.
Features
  • Supports mod packs from Curse
  • Cross platform for Windows and Linux!
  • Easy instance creation, with one click forge support.
  • Import Packs from many different sources
Planned Features
  • Powerful instance manipulation tools.
  • Private Modpack Hosting.
  • Public packs, create mod packs that can be downloaded by anyone with ease.
